# What happens when you type https://www.google.com in your browser and press Enter

## Project Description
This project is a deep dive into the infrastructure of the internet. It covers the journey of a request from the client-side to the server-side, explaining key concepts of the web stack.

## Topics Covered
* DNS Request
* TCP/IP Handshake
* Firewall security
* HTTPS/SSL Encryption
* Load Balancing
* Web, Application, and Database Servers
